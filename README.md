# Nonograms
Solving nonograms
By Tyler Bateman

This repository is a continuous work in progress to solve nonogram puzzles as efficiently as possible.

Inputs files must be in the following format:

number of rows

number of columns

row clues

column clues

 

Example:

5

5

5 2,2 1,1 2,2 5

5 2,2 1,1 2,2 5
